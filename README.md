# Ising_model
L’objectiu d’aquesta pràctica és simular el comportament d’un sistema de spins en una matriu quadrada de 20×20. Cada spin interactua únicament amb els seus quatre veïns més propers. La simulació es durà a terme mitjançant un algorisme Montecarlo que segueix el criteri de Metropolis per decidir si un spin canvia d’orientació.
